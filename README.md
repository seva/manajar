# ManaJar

A Roblox game where players put mana into jars, link them to targets, and cast LLM-generated spells. The spell strength scales based on invested mana vs target resistance.

## Project Constitution

This project follows the **epistegrity** methodology (copied from https://github.com/seva/epistegrity).

See:
- `CLAUDE.md` — session bootstrap
- `METHODOLOGY.md` — rules and protocol
- `ARCHITECTURE.md` — system design
- `IMPLEMENTATION.md` — current phase and tasks
